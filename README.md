FHIR value set editor
=========

## Demo

[Live Demo](http://fhir.github.io/fhir-terminology)

## Installation

```
git clone git@github.com:FHIR/fhir-terminology.git
cd fhir-terminology
npm install
bower install
grunt build # or grunt watch in dev mode
grunt srv
```

Open locahost:8080 in you browser
